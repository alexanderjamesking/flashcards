## Usage

### REPL

`lein repl` or cider jack in (emacs)

```
(dev) ;; switch to the Dev ns

(go) ;; start the server

(reset) ;; reset the server

(halt) ;; stop the server
```

### Running with lein

`lein run` to run the app not via the REPL


### Building the JAR

`lein uberjar`
